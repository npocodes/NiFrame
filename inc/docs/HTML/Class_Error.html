Error Class – (error)

Attributes

·         error – protected – The error dump string holds concatenated error strings

 

Methods

·        __construct() – Initializes attributes to default values

·        Error() – Returns the current error dump string

·        LogError() – Writes an error message to the error log file “error_log”, if E_REPORT constant is set to true then the error message will echo to the browser.

o   ACCEPTS:

§  $msg – the message string to write

§  $filename – the name of the file to write to, default: “error_log”

o   RETURNS:

§  True | False
